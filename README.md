## Insighta Web Interface

The **Insighta Web Interface** provides a powerful, user-friendly dashboard for managing your identity and access workflows. It serves as the central hub for users to authenticate via GitHub, view their count, browse profiles and search various classifications.

---

### Key Features

*   **OAuth2 Integration**: Secure, one-click authentication using GitHub.
*   **Dynamic Profile Management**: View and edit user metadata, roles, and linked accounts.
*   **Real-time Status**: Integrated visual feedback for search profiles.

---

### Technology Stack

| Layer | Technology |
| :--- | :--- |
| **Frontend** | HTML/CSS/JS |
